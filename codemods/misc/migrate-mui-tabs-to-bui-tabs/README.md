# @backstage/migrate-mui-tabs-to-bui-tabs

Migrates MUI Tabs to Backstage UI Tabs.

## Covers

- Standalone Tabs / Tab patterns
- Named / partial barrel imports
- Merging into an existing `@backstage/ui` import

## TODOs / won't-do

- Vertical tabs (`vertical-tabs-todo`)
- Scrollable / centered / custom indicator APIs
