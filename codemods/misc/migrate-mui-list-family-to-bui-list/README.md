# @backstage/migrate-mui-list-family-to-bui-list

Migrates MUI List family primitives to Backstage UI List.

## Covers

- Named / partial barrel List imports
- Simple lists with icons and text-only items
- Merging into an existing `@backstage/ui` import

## TODOs / won't-do

- Interactive list patterns (`interactive-todo`)
- Dense / nested / custom ListItemSecondaryAction layouts
