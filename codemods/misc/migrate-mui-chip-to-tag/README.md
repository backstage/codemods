# @backstage/migrate-mui-chip-to-tag

Migrates MUI 4 `Chip` to Backstage UI `Tag`.

## Covers

- Named / partial barrel imports
- Simple display chips and dynamic string labels
- Merging into an existing `@backstage/ui` import

## TODOs / won't-do

- Clickable / interactive chips (`clickable-chip-todo`, `interactive-chip-todo`)
- Chip avatars, delete icons, and unsupported interactive APIs
