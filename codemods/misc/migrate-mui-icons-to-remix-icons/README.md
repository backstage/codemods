# @backstage/migrate-mui-icons-to-remix-icons

Replaces `@material-ui/icons` imports with `@remixicon/react` equivalents.

## Covers

- Known icon name mappings (simple + multiple icons)
- Extension / slot icon patterns used by Backstage plugins
- Merging into an existing Remix import

## TODOs / won't-do

- Namespace imports (`namespace-import-todo`)
- Unknown / unmapped icons (`unknown-icon-todo`)
- Perfect visual parity for every MUI glyph
